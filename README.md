Xenon— Приложение для отслеживания привычек


Xenon — это интуитивно понятное веб-приложение для формирования и отслеживания полезных привычек. Приложение помогает пользователям отслеживать свой прогресс, визуализировать достижения и сохранять мотивацию для достижения личных целей.

Ключевые особенности
- Создание и управление привычками
- Визуальный трекер с недельной/месячной сеткой
- Статистика прогресса и мотивационные показатели
- Адаптивный дизайн
- Локальное хранение данных

Технологический стек

Основные технологии
- Frontend Framework: Angular 17+ (со standalone компонентами)
- Язык программирования: TypeScript
- Стилизация: CSS, Angular Material
- Реактивное программирование: RxJS
- Хранение данных: LocalStorage
- Тестирование: Jasmine/Karma

Сервисы
1. LocalStorageService — инкапсулирует работу с локальным хранилищем
2. HabitService — управление привычками и бизнес-логика
3. StatsService — расчет статистики и аналитики
4. DateService — утилиты для работы с датами

 Компоненты
- AppComponent — корневой компонент с навигацией
- HabitListComponent — список привычек пользователя
- HabitFormComponent — форма создания/редактирования привычек
- TrackerComponent — календарный трекер привычек
- StatisticsComponent — статистика и графики прогресса

Установка и запуск

Предварительные требования
- Node.js (версия 18 или выше)
- npm (версия 9 или выше)
- Angular CLI (версия 17 или выше)

Функциональность

1. Управление привычками
- Создание новых привычек с настройками
- Редактирование существующих привычек
- Удаление привычек
- Настройка дней повторения

2. Трекинг привычек
- Месячный календарь для отслеживания
- Визуальная индикация выполнения
- Быстрая отметка выполнения за день
- Цветовая кодировка прогресса

3. Статистика и аналитика
- Текущая серия выполнения (streak)
- Общий процент выполнения
- График прогресса по неделям
- История выполнения

4. Мотивационные функции (по мере возможностей)
- Визуальная обратная связь
- Уведомления о прогрессе
- Достижения и награды
- Экспорт статистики

Ключевые тестовые сценарии
1. **Сервис привычек:**
   - Создание новой привычки
   - Обновление существующей привычки
   - Удаление привычки

2. **Сервис статистики:**
   - Расчет текущей серии выполнения
   - Обработка пропущенных дней
   - Расчет процента выполнения

3. **Компоненты:**
   - Валидация форм
   - Взаимодействие с пользователем
   - Корректное отображение данных

даптивный дизайн
